# Expense Tracker

A command line expense tracker built with Python and Pandas — with an upgraded Pro version adding charts and update features.

## Version 1 — expense_tracker_v1.py
Basic expense tracker with add, remove and view features.

## Version 2 Pro — expense_tracker_v2.py
Upgraded version with update function and three Matplotlib charts.

## How to run V1
pip install pandas
python expense_tracker_v1.py

## How to run V2
pip install pandas matplotlib
python expense_tracker_v2.py
